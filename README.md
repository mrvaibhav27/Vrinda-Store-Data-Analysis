# Vrinda Store Data Analysis
## Project Overview
Excel data analysis project to analyze store sales performance, customer behavior, and order trends using pivot tables and dashboards.

## Objectives
- Analyze overall store sales performance
- Understand customer purchasing behavior
- Identify top-selling products and categories
- Track monthly and channel-wise sales trends

## Dataset
- Store sales transaction data
- Includes order date, product category, sales amount, customer details, and sales channel

## Tools & Techniques Used
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Data Cleaning    
- Dashboard Creation

## Analysis Performed
- Monthly sales and trend analysis
- Product and category-wise sales performance
- Customer demographics and purchase pattern analysis
- Sales channel comparison (online)

## Dashboard
Created an interactive Excel dashboard to visualize:
- Total Sales & Orders
- Monthly Sales Trends
- Top Products and Categories
- Customer and Channel-wise Performance

## Key Insights
- Identified best-performing product categories
- Discovered peak sales months and sales channels
- Provided insights to improve sales strategy and inventory planning
